# Enhancing Group Recommendations with Novel CDGRS-LOD Model

Enhancing Group Recommendations with a novel CDGRS-LOD model that leverages Linked Open Data for cross-domain insights. Improves group preference aggregation, tackles data sparsity, and delivers more accurate, diverse, and scalable recommendations for collaborative decision-making scenarios.

## Overview

This project presents a novel **Cross-Domain Group Recommendation System using Linked Open Data (CDGRS-LOD)**, designed to improve the accuracy and relevance of recommendations for groups rather than individuals. By leveraging **Linked Open Data (LOD)**, the model enriches user and item representations across multiple domains, enabling more informed and context-aware recommendations.

The proposed approach addresses key challenges in group recommendation systems, including preference aggregation, data sparsity, and cross-domain knowledge integration. Through semantic relationships derived from LOD, CDGRS-LOD enhances recommendation diversity, scalability, and personalization at the group level.

## System Highlights

* Cross-domain recommendation leveraging Linked Open Data (LOD) for richer knowledge integration
* Advanced group preference aggregation using K-Nearest Neighbors (KNN) and averaging strategies
* Semantic enrichment to enhance recommendation relevance and accuracy
* Scalable and extensible architecture suitable for real-world applications
* Comprehensive evaluation on benchmark datasets using RMSE, MAE, AUC, and ROC curve analysis

## Use Cases

* Group travel planning
* Movie or music recommendation for friends/families
* Collaborative decision-making platforms
* Social and community-based recommendation systems
